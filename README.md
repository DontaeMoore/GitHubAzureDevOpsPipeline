# GitHubAzureDevOpsPipeline

Simple starter project for GitHub and Azure DevOps.

## Run locally

```bash
py -3 app.py
```

Expected output:

```text
Hello, world!
```

## Azure DevOps pipeline

This repo includes `azure-pipelines.yml`.

In Azure DevOps:

1. Create a new pipeline.
2. Choose GitHub as the source.
3. Select this repository.
4. Choose Existing Azure Pipelines YAML file.
5. Select `/azure-pipelines.yml`.
